# Nada Zero

Shortwave numbers station simulator for iOS.

Nada Zero generates continuous shortwave radio atmospherics—static, drone, and a voice reading number groups encoding public domain folk and fairy tales from [Andrew Lang's Fairy Books](https://www.gutenberg.org/ebooks/30580)—creating an ambient numbers station experience you can tune and listen to.

- **Privacy-first** — no data collection, no accounts, no analytics, no tracking.
- **Fully offline** — all audio is generated on-device in real time. No network access required.
- **Configurable** — adjust voice, noise, and drone levels to taste.
